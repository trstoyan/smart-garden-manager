Tools Tutorial

Use this checklist to run setup, daily care, and automation tools with confidence.

1. Daily Workflow (Web UI)
  1. Open Guided Start for first-time setup.
  2. Use Setup Center to create gardens, plant types, and groups.
  3. Open Plants and add your plants.
  4. Register hardware in Devices.
  5. Review telemetry in Sensor Readings.
  6. Open Rules Center for group/plant care rules.
  7. Open Rule Coverage to find plants/groups missing active rules.
  8. Plan and complete tasks from Care Calendar.
  9. Run queue tools in Device Actions and Notifications Center.
2. Device Data Ingest

Send readings to POST /api/sensor-data/ with device key and optional idempotency key:

curl -X POST http://localhost:8000/api/sensor-data/ \
  -H "Content-Type: application/json" \
  -H "X-Device-Key: <device_api_key>" \
  -H "X-Idempotency-Key: reading-2026-02-27T23:00:00Z" \
  -d '{"device_id":"ESP32_001","temperature":23.5,"humidity":52,"soil_moisture":480,"light":190}'
3. Automation + Dispatch Commands
cd smart_garden
python manage.py evaluate_automations
python manage.py process_device_actions --batch-size 100 --max-attempts 6
4. Notification Commands
cd smart_garden
python manage.py generate_upcoming_notifications --days 2 --daily-limit 12
python manage.py process_notifications --batch-size 100 --max-attempts 6
5. Pest Follow-up Scheduling
cd smart_garden
python manage.py schedule_pest_followups --days 3
